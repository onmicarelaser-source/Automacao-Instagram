\# 🤖 Automação de Posts e Stories Instagram



\## 📋 Sobre o Projeto

Automação diária de posts e stories no Instagram utilizando artes do Canva.



\## 🚀 Funcionalidades

\- Publicação automática de posts às 8h

\- Publicação automática de stories às 18h  

\- Integração com API do Instagram

\- Sistema de logs e monitoramento



\## 🛠️ Tecnologias

\- Python 3.9+

\- Instagram Graph API

\- Canva API

\- Schedule (agendamento)
